RIS: - Rider translator for RUS and Windows (RIDER.SAV & RIDER.EXE)

Rider is a C-based procedural language. Rider translates into both
PDP-11 and Windows C allowing programs to be developed to run on
both systems. Rider has workarounds for some DECUS-C limitations.

RIS:	Shared sources
RIB:	Windows binaries
RIP:	RUST binaries

Rider source files have filetype ".r" and translate to CC ".c" files
Rider definition files have filetype ".d" and translate to CC ".h" files
